# Synergy Addons v3.0.0 (Discontinued)

**LEGAL NOTICE:** The plugin is in no way affiliated with Edupoint and is discontinued, so it should not be used.

**POLICY:** Synergy Addons does not collect any user data. The only reason it asks for page access is to change the visuals of Synergy. It is also COPPA and FERPA compliant.

## Features
- Dark Mode
- Student ID Hider (BETA)

## Installation
This plugin is no longer available for installation.
